# Password Generator CLI

CLI-генератор криптографически безопасных паролей на Node.js.

## Возможности

- Криптографически безопасный рандом (`crypto.randomBytes`, не `Math.random`)
- Настройка длины пароля (4-128 символов)
- Выбор набора символов (строчные, заглавные, цифры, спецсимволы)
- Исключение похожих символов (`0`, `O`, `l`, `1`, `I`)
- Гарантия хотя бы одного символа из каждой выбранной категории
- Оценка сложности пароля (энтропия в битах)
- Генерация до 100 паролей за раз

## Установка

```bash
npm install
```

## Использование

### Базовое
```bash
node src/index.js
```

### С опциями

```bash
# Пароль длиной 24 символа
node src/index.js --length 24

# Без спецсимволов
node src/index.js --no-symbols

# 5 паролей с отображением энтропии
node src/index.js --count 5 --entropy

# Только цифры и строчные, без похожих символов
node src/index.js --no-uppercase --no-symbols --exclude-similar

# Тихий режим (только пароль, для пайпов)
node src/index.js --length 32 --quiet
```

## Флаги

| Флаг | Описание | По умолчанию |
|------|----------|-------------|
| `--length N` | Длина пароля (4-128) | 16 |
| `--no-lowercase` | Исключить строчные буквы | false |
| `--no-uppercase` | Исключить заглавные буквы | false |
| `--no-digits` | Исключить цифры | false |
| `--no-symbols` | Исключить спецсимволы | false |
| `--exclude-similar` | Исключить похожие символы (0, O, l, 1, I) | false |
| `--count N` | Количество паролей (1-100) | 1 |
| `--entropy` | Показать расчёт энтропии | false |
| `--quiet` | Вывести только пароль | false |
| `--help`, `-h` | Показать справку | — |

## Примеры вывода

### Обычный режим
```
Сгенерированный пароль:
kR9#mP2$vL5@nQ8!

Длина: 16 | Набор символов: 94
```

### С энтропией
```
Сгенерированный пароль:
xT7&bN4!wK9@mR2#

Энтропия: 104.87 бит
Сложность: Очень сильный

Длина: 16 | Набор символов: 94
```

### Несколько паролей
```
Пароль 1:
aB3!xY7@kL9#mN2$

Пароль 2:
qW8&rT5%uI1^oP4*

Длина: 16 | Набор символов: 94
```

### Тихий режим
```
kR9#mP2$vL5@nQ8!
```

## Уровни сложности

| Энтропия | Уровень |
|----------|---------|
| < 36 бит | Слабый |
| 36-59 бит | Средний |
| 60-79 бит | Сильный |
| >= 80 бит | Очень сильный |

## Безопасность

- Используется `crypto.randomBytes()` с rejection sampling для равномерного распределения
- Нет `Math.random()` нигде в коде
- Валидация всех входных данных

## Требования

- Node.js >= 14
